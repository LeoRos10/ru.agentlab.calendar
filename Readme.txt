
Учебный проект клиента календаря Google на Eclipse RCP с использованием JavaFX


1. Скачайте Eclipse Oomph
Скачать Eclipse Installer (Oomph) по ссылке: https://wiki.eclipse.org/Eclipse_Installer

2. Установить Eclipse Oomph и запустить

3. Переключиться в режим Advanced

4. В окне Products настроить базовый продукт
Выбрать настройки:
Product: Eclipse IDE for Java Developers
Product Version: Latest Release (Mars)
Java: Java 8

5. В окне Projects добавить файлы конфигурации проектов
Выделить папку Github.com
При помощи кнопки с зеленым знаком «плюс» добавить URL-ссылки файлы конфигурации (в зависимости от того, что Вы разрабатываете, см. список ниже):

https://raw.githubusercontent.com/agentlab/ru.agentlab.calendar/master/setup/ru.agentlab.calendar.setup/ru.agentlab.calendar.setup
 
6. В окне Projects выбрать проекты системы, которые вы будете дорабатывать
После добавления нужного(ых) файла(ов) конфигурации проектов, необходимо выбрать проекты в окне Projects.
Сначала обязательно выберите родительский проект с общими настройками. Обычно он называется XXX Setup.
Потомвыберите
Например, для разработчика платформы 

7. Настройка Eclipse IDE и проектов
После нажатия кнопки Finish на последнем окне Eclipse Installer произойдет следующее:
Скачивание выбранного продукта Eclipse и дополнительных плагинов Eclipse из настроек проектов
Настройка Eclipse IDE
Клонирование репозитариев и импорт проектов из них в Eclipse IDE
Скачивание бинарных зависимостей проектов и формирование целевой платформы

8. Обновление maven-зависимостей и принудительная пересборка системы
После завершения настройки необходимо выделить любой maven-проект (с буквой М) и из контекстного меню выбрать Maven->Update Project...
В появившемся окне нажать Select All и OK
